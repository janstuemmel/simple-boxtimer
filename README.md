# Simple Boxtimer

![logo](./assets/playstore-functional.png)

A simple timer for boxing. Written with react native, works on android and the web.

## Development

```sh
# install expo
npm i -g expo-cli

# install deps
npm i

# run application
npm run android
npm run web
```

## Build

```sh
# build for android or web
npm run build:android
npm run build:web
```
